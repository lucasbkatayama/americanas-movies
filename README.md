# Americanas Filmes

Americanas Filmes é um projeto em React para buscar filmes pela OMDB api.

## Instalação

Use o [npm](https://www.npmjs.com/) para instalar as dependencias do Americanas Filmes.

```bash
npm install
```
## Uso

```bash
npm run start
```

## Considerações
Não está completo e está em desenvolvimento.
